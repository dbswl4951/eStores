eStores
============

물품 등록, 조회, 구매가 가능한 웹 페이지 입니다.

- 사용자 : 원하는 물품을 장바구니에 추가하거나 구매 가능
- 관리자 : 사진 포함 한 물품 올리기/삭제 가능  


## 사용 기능, 기술

- Spring MVC/Security, REST API
- Hibernate, MySQL
- AWS EC2
